# CodeAtlas

CodeAtlas is an open-source e-learning platform that turns curated YouTube playlists into structured, enrollable programming courses.

Users can:
- Browse courses by language or topic
- Enroll and track progress through a personal dashboard
- Learn from high-quality content â€” no paywalls

## Core Principles
- Free education
- Open-source collaboration
- Practical, real-world learning

## Tech Stack
- **Backend:** FastAPI (Python)
- **Frontend:** HTML, CSS, vanilla JS
- **Templating:** Jinja2
- **Data source:** YouTube API

## Project Structure
```
templates/       # Jinja2 HTML templates
static/css/      # Stylesheets
static/js/       # Client-side scripts
```
